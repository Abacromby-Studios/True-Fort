How to Connect

    1. Open Minecraft Java Edition
    2. Click on "Multiplayer"
    3. Click "Add Server"
    4. Enter "TrueFort" as the server name
    5. Enter truefort.minehut.gg as the server address
    6. Click "Done" and then select the server from your list
    7. Click "Join Server"
    1. Open Minecraft Bedrock Edition
    2. Click on "Play"
    3. Navigate to the "Servers" tab
    4. Scroll down and click "Add Server"
    5. Enter "TrueFort" as the server name
    6. Enter truefort.bedrock.minehut.gg as the server address
    7. Enter 19132 as the port
    8. Click "Save" and then select the server from your list
